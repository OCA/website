=========================
Website Blog - Categories
=========================

This module provides the ability to categorize blog posts.

Usage
=====

To use this module:
-------------------

* Go to `Website Admin` => `Blog Categories` to create blog categories.
* Enable category browsing in Blog website customization options
* Category browser is displayed in blog sidebar
