==========================
Website Snippet Breadcrumb
==========================

This module allows users to add a breadcrumb snippet into the website

Installation
============

To install this module, you need to:

 * click on install

Usage
=====

To use this module, you need to:

 * go to into website
 * edit/insert block
 * into content: select breadcrumb
 * save then you can see your breadcrumb

Note that the breadcrumb is based on the menu hierarchy: Only pages referenced
into a menu will trigger the computation of the breadcrumb
