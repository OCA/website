This module remove Odoo branding on website:

* Remove "Create a free website with Odoo" from footer

**Table of contents**

.. contents::
   :local:

Usage
=====

#. Go to your home page.
#. Enable or disable *Customize > Footer Copyright > Remove Odoo Promotional
   Link* at your will. It should be enabled by default.
