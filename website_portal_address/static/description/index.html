Website Portal Contact Address


Multiple Address can be created from user account


This module was written to add the functionality of adding multiple Address for contact and user can edit/delete or create new address. The Module provides types of address like, shipping address, invoice address, other address, contact.

This Module depends on OCA module which is "website_portal_contact". you can get this module from below link: https://github.com/OCA/website/tree/9.0/website_portal_contact
