Enhance Website Portal

This module extends the functionality of website portal module and adds the support of displaying/adding image of user. Module also enhanced the view of my account menu in website and user can be able to change the image directly by clicking on the user image.

View Profile

User can view profile by clicking on the pencil icon. Users detail will displayed as shown in the image.

Edit Profile

User can edit profile by clicking on the user icon. Users detail will displayed as shown in the image.
