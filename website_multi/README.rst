=============
Multi Website
=============

This module allows you to build and manage multiple website inside the same
Odoo database and company.


Installation
============

This addon is not compatible with 'dbfilter' configuration parameter.
So, you should have only one database to avoid database selector page for
new visitors and other multiple databases issues.


Usage
=====

New top menu "Website Admin" will appear with these options:

* Website
    * Websites: List of
    * Pages
    * Menus
    * Snippets
* Template
    * Templates
    * Website wizard
* Theme
    * Themes
